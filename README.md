# Portfólio — João Victor Oliveira

Portfólio pessoal com foco em web design aplicado, interface, layout, identidade visual aplicada e desenvolvimento front-end. Os estudos de caso principais são BEE1 e Barraca do Pezão.

## Posicionamento

João Victor atua principalmente na interseção entre web design e implementação: traduz referências, identidades existentes e objetivos comerciais em interfaces responsivas e publicadas. O portfólio não o apresenta como designer dedicado à criação completa de marcas ou protótipos extensos no Figma.

## Tecnologias

- HTML semântico
- CSS responsivo
- JavaScript
- GSAP + ScrollTrigger
- WordPress e Elementor nos projetos apresentados

## Executar localmente

Abra `index.html` diretamente no navegador ou use um servidor local:

```bash
python3 -m http.server 8080
```

Depois acesse `http://localhost:8080`.

## Publicação no GitHub Pages

1. Envie os arquivos para a branch principal.
2. No GitHub, abra **Settings → Pages**.
3. Em **Build and deployment**, selecione **Deploy from a branch**.
4. Selecione a branch `main` e a pasta `/root`.
